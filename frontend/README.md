# Hedge v2 Frontend
这是App的前端子项目。它的目标运行容器为App(`Electron`)及浏览器(`Web`)。

## Technology Stack
* `node >= 14.14.0`
* `vue 3.x`(`vue-cli`, `vue-router`, `jsx`)
* `typescript`
* `scss`
* `bulma`
* `font-awesome 5.x`

## Development & Debug
### Deploy
```sh
yarn    # 安装全部依赖
```
### Development
```sh
yarn dev    # 打开HMR开发服务器
```
### Build
```sh
yarn build  # 编译，输出在dist目录下
```
