# Hedge v2 Command Line Interface
这是App的命令行工具子项目。它不是App运行的必备组件，而是在CLI环境下的服务控制、功能实现和额外CLI工具补充。

## Technology Stack
* `Python >= 3.9`

## Development & Debug
### Deploy
```shell
virtualenv venv
source venv/bin/active
pip3 install -r requirements.txt  # 安装全部依赖
```

## Build
